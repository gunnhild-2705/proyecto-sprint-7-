# proyecto-sprint-7-
# Análisis de vehículos usados en EE. UU.

## Descripción del proyecto
Este proyecto consiste en el desarrollo de una aplicación web interactiva creada con Streamlit para el análisis de un conjunto de datos de vehículos usados en Estados Unidos.

La aplicación permite explorar visualmente los datos mediante gráficos interactivos, facilitando la comprensión de la relación entre distintas variables del conjunto de datos, como el precio y el kilometraje de los vehículos.

## Funcionalidad de la aplicación web
La aplicación web proporciona las siguientes funcionalidades:
- Visualización de un histograma para analizar la distribución de una variable numérica del conjunto de datos.
- Generación de un gráfico de dispersión interactivo que muestra la relación entre el kilometraje y el precio de los vehículos.
- Interacción mediante botones que permiten al usuario decidir qué visualización mostrar.

## Tecnologías utilizadas
- Python
- Pandas
- Streamlit
- Plotly Express

## Objetivo
El objetivo principal de esta aplicación es demostrar el uso de herramientas de visualización de datos y el desarrollo de aplicaciones web interactivas como parte del análisis exploratorio de datos.
